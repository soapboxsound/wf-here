PIN JAVASCRIPT FUNCTION

In /js/pins.js create and export a function called createPin that accepts an options object with these properties: type (either "standard" or "featured"), size (either "lg", "md", or "sm"), name (string, optional), showLabel (boolean, optional).

The function returns a div element with class "map-pin" containing the SVG element. If showLabel is true and name is provided, add a span below the SVG with class "pin-label" containing the name text.
